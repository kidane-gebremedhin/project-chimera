## Functional Requirements (User Stories)

### Agent Capabilities

* **As an Agent**, I need to identify high-growth topics and fetch trending topics relevant to my niche so that I can generate timely content.
* **As an Agent**, I need to remember past high-performing posts to maintain consistency and improve engagement.
* **As an Agent**, I need to generate text, images, or video using approved tools so that I can publish content autonomously.
* **As an Agent**, I need to disclose that I am AI-generated when asked, to remain compliant and transparent.
* **As an Agent**, I need to manage a non-custodial wallet to pay for API credits and receive sponsorship revenue.

### Planner Responsibilities

* **As a Planner**, I need to decompose high-level goals into executable tasks.
* **As a Planner**, I need to reprioritize tasks when context changes (trends, failures, budget limits).

### Judge & Safety

* **As a Judge**, I need to validate content against persona, safety, and confidence thresholds.
* **As a Judge**, I need to escalate risky or low-confidence outputs to a human reviewer.

### Human Operator

* **As a Human Reviewer**, I need to approve or reject flagged content quickly without technical context.
* **As an Operator**, I need visibility into agent status, queue depth, and financial health.

## Core Workflow
* **Discovery:** Identify trending hashtags.
* **Creation:** Generate video/audio assets.
* **Distribution:** Publish to social platforms via MCP tools.
